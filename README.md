# react-taskslist

## 1. Description :

- This is react web application for task list.
- Created using react, redux and thunk.

## 2. Prerequisite:

- Node Js
- React Js

## 3. Install

```bash
yarn install
```

## 4. Start

```bash
 yarn start
```

## 5. Run

- Click [here](http://localhost:3000) to run in browser.
